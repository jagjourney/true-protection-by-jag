# True Protection Browser Extension

Web threat protection powered by [True Protection by Jag](https://tpjsecurity.com).

## Features

- Block known malicious websites
- Phishing detection
- Dangerous download warnings
- Real-time threat feed updates
- Works with Chrome, Edge, and Firefox

## Install

Available on:
- [Chrome Web Store](https://tpjsecurity.com/download) (coming soon)
- [Firefox Add-ons](https://tpjsecurity.com/download) (coming soon)
- [Edge Add-ons](https://tpjsecurity.com/download) (coming soon)

## Development

1. Clone this repo
2. Go to `chrome://extensions`
3. Enable Developer mode
4. Click "Load unpacked" and select this folder
5. Make changes and reload
