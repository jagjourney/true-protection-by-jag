# True Protection by Jag - Browser Extension

Real-time web protection for Chrome and Brave browsers.

## Features

- **Phishing Detection** - Identifies fake login pages, typosquatting domains, and homoglyph attacks
- **Cryptojacking Blocker** - Blocks cryptocurrency mining scripts from running in your browser
- **Malware Protection** - Detects malicious iframes, credential harvesting forms, and suspicious scripts
- **Domain Blocklist** - Continuously updated list of known malicious domains
- **Download Scanning** - Warns before downloading files from untrusted sources
- **Password Manager** - Secure credential autofill powered by the True Protection desktop vault
- **Activity Logging** - View your protection activity on the tpjsecurity.com dashboard

## Installation

### From the Store (Recommended)

Coming soon to the Chrome Web Store and Brave Web Store.

### Manual Install (Developer Mode)

1. Download or clone this repository
2. Open your browser and go to `chrome://extensions` (or `brave://extensions`)
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** and select this directory
5. Log in with your tpjsecurity.com account from the extension popup

## Requirements

- Chrome 102+ or Brave (latest)
- A True Protection account at [tpjsecurity.com](https://tpjsecurity.com)
- For password manager and advanced features: True Protection desktop app installed
