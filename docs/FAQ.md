# Frequently Asked Questions

## Is True Protection really free?

Yes. The core protection engine (firewall, antivirus, anti-rootkit, HIPS, NIDS) is completely free. Advanced AI-powered features (JagAI) require a paid subscription starting at $4.99/month.

## What platforms are supported?

Windows 10/11, macOS (Intel and Apple Silicon), and Linux (Ubuntu, Fedora, Arch, and most distributions).

## How do I install True Protection?

Download from [tpjsecurity.com/download](https://tpjsecurity.com/download). Run the installer for your platform.

## How do I update?

True Protection auto-updates by default. You can also check manually: `tpj update --check`

## What is JagAI / Advanced Jag Tools?

JagAI is our AI-powered threat analysis platform. It provides 24/7 monitoring, threat hunting, automated incident response, and priority signature updates. Available as a monthly subscription.

## How do I report a bug?

Open an issue on this GitHub repository using the bug report template.

## How do I report a security vulnerability?

Email security@jagjourney.com. Do NOT open a public issue. See SECURITY.md.

## Can I contribute to the project?

Yes! You can contribute YARA detection rules and browser extension improvements. See CONTRIBUTING.md.
