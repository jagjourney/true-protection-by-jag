# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in True Protection, **please do NOT open a public issue.**

Email: **security@jagjourney.com**

Include: description, steps to reproduce, affected version, platform, severity assessment.

### What to Expect

- **Acknowledgment:** Within 48 hours
- **Assessment:** Within 5 business days
- **Resolution:** Critical fixes within 30 days
